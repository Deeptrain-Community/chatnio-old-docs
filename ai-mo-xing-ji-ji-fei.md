## 最低余额限制

为了防止超出余额的部分过多，同时为了防止滥用，我们定义了特定模型的最低余额，当大于等于该余额时，可发起请求：

> 对于按照次数计费模型，最低余额为单次调用费用。
>
> 对于按照 Token 计费模型，最低余额为 1K 输入 Token 费用 + 1K 输出 Token 费用。

{% hint style="success" %}
免费模型均无最低余额限制
{% endhint %}
